Data-aanbod ()
Er is sprake van data als handelingen, feiten of informatie digitaal worden weergegeven. De data is op enigerlei wijze persistent opgeslagen in een of meer fysieke databronnen. De ontsluiting van de data uit de fysieke bron, gebeurt door middel van dataservices. Een dataservice is een door de data-aanbieder samengesteld mechanisme dat na activering door een applicatie, toegang geeft tot bepaalde data uit een of meer fysieke databronnen. Data-afnemers kunnen als ze daartoe gerechtigd zijn, de dataservices activeren door deze met een eigen applicatie aan te roepen en de daaruit verkregen data gebruiken voor hun eigen doelen in hun eigen context.
